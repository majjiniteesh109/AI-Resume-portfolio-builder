AI Resume, Portfolio & Cover Letter Builder

1. Project Overview:
A brief description of the project:
    Many students struggle to present their skills and projects in a professional format. This project provides an AI-powered solution to automatically generate resumes, portfolios, and cover letters based on student inputs, improving their chances of securing jobs and internships.

2. Features
Main features:
    1.Generate professional resumes using AI.
    2.Create personalized portfolios.
    3.Build cover letters tailored to a userâ€™s details.
    4.Download outputs as PDF files.
    5.All inputs are provided by the user through a simple Streamlit interface.

3. Installation Instructions
Step-by-step instructions to run the project:
    1. pip install -r requirements.txt
    2.Set up API Key
        Add your Google Generative AI API key in the respective code files:
         GOOGLE_API_KEY = "YOUR_GOOGLE_API_KEY_HERE"
    3.Run the Streamlit app
        python -m streamlit run main.py

4. Usage
    1.Open the Streamlit app in the browser.
    2.Enter your personal details, education, skills, projects, certifications, and achievements.
    3.Click buttons to generate AI-powered Resume, Portfolio, or Cover Letter.
    4.Download the generated PDF files.
